# 💰 Cash Flow Dashboard

A responsive expense tracking dashboard built using **HTML, CSS, and Vanilla JavaScript**. The application enables users to manage salary, record expenses, visualize spending patterns, and generate reports.

## 🚀 Features

### 📊 Dashboard

* Real-time salary, expense, and remaining balance calculation.
* Dynamic summary cards.
* Responsive user interface.

### 💵 Expense Management

* Add expense name and amount.
* Delete individual expenses.
* Automatic balance updates.

### 📈 Analytics

* Interactive Pie Chart using Chart.js.
* Visual comparison of expenses and remaining balance.

### 🌙 Theme Support

* Dark Mode / Light Mode toggle.
* Theme persistence using LocalStorage.

### 💾 Data Persistence

* Saves salary and expenses using LocalStorage.
* Data remains available after page refresh.

### ⚠️ Budget Alerts

* Warning banner when overspending occurs.

### 📄 PDF Report Generation

* Generate and download expense reports using jsPDF.

### 💱 Currency Conversion

* Supports multiple currencies with dynamic conversion.

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js
* jsPDF
* LocalStorage API
